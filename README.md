# SillyTavern 小说章节导入助手
SillyTavern 专用扩展，支持一键导入本地小说TXT文件，自动按章节拆分，通过斜杠命令批量发送到聊天对话框，轻松实现小说内容的快速导入。

## 功能特性
- 📖 支持导入本地UTF-8编码的TXT小说文件
- 🔧 可自定义章节拆分正则表达式，适配所有小说的章节标题格式
- 📋 实时章节预览，支持单选、多选章节导入
- ✨ 自定义斜杠命令模板，支持角色名、章节内容变量替换
- 🚀 批量自动发送，内置防刷屏延迟，稳定不报错
- 💾 用户配置自动持久化，重启不丢失

## 安装方法
### 在线安装（推荐）
1. 打开 SillyTavern，进入「扩展插件」页面
2. 选择「安装扩展」，粘贴本仓库的Git地址
3. 点击「安装」，安装完成后启用扩展
4. 刷新页面，侧边栏扩展区会出现「小说导入」按钮

### 手动安装
1. 下载本仓库完整代码，解压到 SillyTavern 安装目录下的 `scripts/extensions/third-party` 文件夹
2. 重启 SillyTavern 后端服务
3. 进入「扩展插件」页面，找到「小说章节导入助手」，启用扩展
4. 刷新页面即可使用

## 使用方法
1. 点击侧边栏扩展区的「小说导入」按钮，打开导入面板
2. 点击「选择小说TXT文件」，上传本地的小说文件
3. （可选）修改「章节拆分正则表达式」，适配你的小说章节格式
4. （可选）修改「发送命令模板」，自定义发送逻辑
5. 点击「解析章节」，预览拆分后的章节列表
6. 勾选需要导入的章节，点击「导入选中章节」或「导入全部章节」
7. 等待导入完成，章节内容会自动发送到聊天对话框

### 命令模板说明
- `{{char}}`：自动替换为当前选中的角色名称
- `{{content}}` / `{{pipe}}`：自动替换为章节正文内容
- 你提供的命令适配：将模板改为 `/input 请输入{{char}}的动作或台词：| /sendas name={{char}} {{pipe}}` 即可
- 默认模板：`/sendas name={{char}} {{content}}`，直接使用当前角色身份发送章节内容，适合批量导入

## 前置要求
- SillyTavern 版本 >= 1.12.0
- 启用内置的 `/sendas` 斜杠命令（默认已启用）
