# Always Remember Me - SillyTavern 小说章节导入助手
SillyTavern 专用扩展插件，支持一键导入本地小说TXT文件，自动按章节拆分，通过自定义斜杠命令批量发送到聊天对话框，轻松实现小说内容的快速导入。

## 功能特性
- 📖 支持导入本地UTF-8编码的TXT小说文件
- 🔧 可自定义章节拆分正则表达式，适配所有小说的章节标题格式
- 📋 实时章节预览，支持单选、多选、全选章节导入
- ✨ 完全自定义斜杠命令模板，支持{{char}}、{{pipe}}、{{content}}变量替换
- 🚀 批量自动发送，内置防刷屏延迟配置，稳定不报错
- 💾 用户配置自动持久化，重启不丢失
- ⌨️ 内置快速斜杠命令，一键唤起文件选择
